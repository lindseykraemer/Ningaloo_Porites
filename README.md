# Ningaloo_Porites
Past thermal history and heat stress shape patterns of coral-algal symbioses in massive *Porites*

A description of each branch within Ningaloo_Porites:
* bbduk_dada2_phyloseq : bioinformatics workflow from sequencing data --> diversity and differential abundance analyses
* CRW_data : extraction of Coral Reef Watch data for the study site (sourced from NOAA's CRW) https://coralreefwatch.noaa.gov/
* maps : generating maps of the study site that incorporate temperature metrics (CRW_data)
* phyloseq_subsets : analyses utilising VST data
  
Sequencing data are available on NCBI Sequence Read Archive under the Bioproject accession number PRJNA1433380.
